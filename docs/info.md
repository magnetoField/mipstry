## How it works

Simple MIPS processor with reduced instruction set 8 bit bus for external memory. An implementation of a 32-bit single cycle MIPS processor in Verilog. 
This version of the MIPS single-cycle processor can execute the 
following instructions: add, sub, and, or, slt, lw, sw, beq, addi, and j.
